# Wireless, Monitoring, Automation & Hybrid Cloud Networking Lab

## Overview
This project demonstrates a hands-on lab focused on enterprise wireless networking,
network monitoring, automation, and hybrid cloud connectivity.

## Wireless Design
- Cisco 9130AX and Aruba AP-515 concepts
- WLC-based architecture
- RF planning, SSIDs, VLAN mapping
- 802.1X authentication

## Monitoring & Observability
- SNMP, Syslog, NetFlow
- SolarWinds and Wireshark
- Performance visualization using Grafana

## Automation
- Configuration backup and compliance checks using Ansible
- Version control using Git

## Hybrid Cloud Networking
- AWS Transit Gateway
- Azure VPN Gateway
- Site-to-site IPSec VPN connectivity

## Cloud Security
- AWS Security Groups and NACLs
- Azure NSGs

## Troubleshooting
- Ping, traceroute, log correlation
- Packet capture and analysis

## Disclaimer
This is a hands-on academic and lab-based project created for learning and demonstration purposes.
